## [0.1.0] - 2026-05-10
### Added
- Initial project setup and design for S&P 500 scraper.
- `get_sp500` API for retrieving S&P 500 constituents.
- Support for multiple return formats (list, dict).
- Support for filtering specific fields (symbol, name, sector, etc.).
- Basic unit tests for scraper and API.
- Professional README and LICENSE.
